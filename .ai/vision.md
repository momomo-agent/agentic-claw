# Vision — agentic-claw

AI agent runtime: wire agentic-core + agentic-memory into a living agent.

- **One function**: `createClaw({ apiKey })` → chat, learn, recall
- **CLI included**: `claw "question"` or `claw -i` for REPL
- **Zero dependencies**: peers only (agentic-core, agentic-memory)
- **Multi-session**: isolated conversations, shared knowledge
- **Lifecycle**: heartbeat, schedule, events

Part of the agentic family — building blocks for AI.
