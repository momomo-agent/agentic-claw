Five lines to an agent

No framework. No boilerplate. Just createClaw and go.

app.js
const { createClaw } = require('agentic-claw') const claw = createClaw({ apiKey: 'sk-...' }) const r = await claw.chat('What is quantum computing?') await claw.chat('Tell me more') // remembers context
Multi-session + Knowledge
const claw = createClaw({ apiKey: 'sk-...', knowledge: true, }) // Teach it — knowledge shared across sessions await claw.learn('docs', longDocument) // Chat — auto: recall → LLM → store await claw.chat('What does the doc say about X?') // Multi-session — isolated conversations const alice = claw.session('alice') const bob = claw.session('bob') await alice.chat('My name is Alice') await bob.chat("What's Alice's name?") // doesn't know
Lifecycle — heartbeat + schedule
// Heartbeat — periodic check-in claw.heartbeat(() => { console.log('Still alive') }, 60000) // Schedule — recurring tasks claw.schedule('5m', () => { console.log('Every 5 minutes') }) // Events claw.on('token', d => process.stdout.write(d.text)) claw.on('error', e => console.error(e))

What's inside

🧠 Auto Memory

Every conversation automatically stored, trimmed, and managed. No manual history threading. Just chat.

🔍 Knowledge Recall

Learn documents once, recall relevant context on every chat. Local TF-IDF or OpenAI embeddings.

👥 Multi-Session

Isolated conversations, shared knowledge. Each session has its own history but taps the same knowledge base.

💓 Heartbeat + Schedule

Periodic callbacks and scheduled tasks. Your agent stays alive and can do work between conversations.

🔌 Streaming

Token-by-token streaming with event callbacks. Proxy support for GFW and CORS. Works everywhere.

📦 Zero Dependencies

Peers: agentic-core + agentic-memory. No transitive deps. No build step. Works in Node and browser.
