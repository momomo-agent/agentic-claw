# Library

const { createClaw } = require('agentic-claw')

const claw = createClaw({
  apiKey: 'sk-...',
  provider: 'anthropic',
  knowledge: true,
})

// Teach it something
await claw.learn('docs', longDocument)

// Chat — auto: memory → recall → LLM → store
const result = await claw.chat('What does the doc say?')

// Multi-session
const alice = claw.session('alice')
const bob = claw.session('bob')
await alice.chat('Hi')  // isolated conversation

# Features

🧠 Auto Memory

Conversations automatically stored and trimmed. Multi-session with isolation.

🔍 Knowledge

Learn documents, recall relevant context. Local TF-IDF or OpenAI embeddings.

🛠 Tools

Custom tool registration. Built-in search and code execution.

⚡ Streaming

Real-time token streaming with proxy support for GFW.

💓 Heartbeat

Periodic callbacks and scheduled tasks. Your agent stays alive.

📦 Zero Deps

Peers: agentic-core + agentic-memory. No transitive dependencies.
